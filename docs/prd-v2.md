## 一、文档总则

### 1.1 文档目的

本文档为全球跨境生活一站式 AI 智能顾问平台的官方执行依据，明确产品顶层定位、核心架构、功能模块、业务流程、内容体系、技术要求及落地规划。

**目标**：一套产品架构，适配全球任意国家/语种/人群，用 AI 全面替代全球传统留学、移民、安家、职场、法律类中介/私人顾问，打造标准化、普惠化、全球化的跨境生活咨询服务体系。

### 1.2 核心原则

| 原则 | 说明 |
|------|------|
| **通用性** | 产品框架、类目结构、顾问角色全球通用，无需重构即可复制至任意国家 |
| **独立性** | 分国家/语种/领域知识库完全隔离，问答精准不串线 |
| **轻量化** | 单机可部署，模块化迭代，前期无需复杂分布式架构 |
| **商业化闭环** | AI 自助咨询 + 真人顾问对接双模式，兼顾普惠与盈利 |
| **全人群覆盖** | 无种族、无语言、无地域限制，服务所有跨境流动人群 |

### 1.3 产品核心定位

> **全球首个覆盖全语种、全国家、全人群的跨境生活 AI 顾问生态平台**

以 RAG 多知识库为技术核心，打造角色化 AI 顾问体系，覆盖跨境留学、移民身份、落地安家、职场就业、法律权益全场景。替代传统中介信息差、高收费、服务碎片化痛点，为全球跨境留学/移民/旅居/工作人群提供 7×24 小时标准化、专业化、免费/低成本咨询服务，同时搭建全球真人专业顾问入驻对接平台，实现 AI + 人工全场景服务覆盖。

### 1.4 产品核心价值

1. **颠覆全球传统中介行业**，消除跨境服务信息差，降低咨询成本
2. **实现跨境生活咨询服务标准化、统一化**，杜绝服务口径不一
3. **7×24 小时不间断服务**，突破地域、时间、人工服务限制
4. **知识永久沉淀迭代**，平台随数据积累持续优化，服务能力持续升级
5. **全国家、全语种快速复制**，打造全球化跨境服务统一入口

## 六、RAG 知识库系统需求

### 6.1 知识库架构

```
全球知识库
├── 总库：全球跨境基础常识库
│
├── 国家分库 (Canada)
│   ├── 🎓 教育求学子库 (edu-*)
│   ├── 🛂 移民身份子库 (imm-*)
│   ├── 🏠 安家落地子库 (life-*)
│   ├── 🏥 医疗健康子库 (health-*)
│   ├── 💰 财税金融子库 (fin-*)
│   ├── 💼 职场就业子库 (career-*)
│   └── ⚖️ 法律权益子库 (legal-*)
│
├── 国家分库 (USA)
│   └── ...同上 7 大类结构
│
└── 国家分库 (UK / Australia / ...)
    └── ...同上 7 大类结构
```

**ChromaDB 命名规则**：

```
{country}_{slug}                → 国家+角色知识库
  例: ca_edu-school-planning       加拿大选校顾问
  例: us_edu-school-planning       美国选校顾问

user_{userId}_{country}_{slug}  → 用户私有文档
  例: user_abc123_ca_life-rental   用户abc的加拿大租房文档
```

### 6.2 知识库内容要求

| 要求 | 说明 |
|------|------|
| **真实性** | 全部内容为当地官方政策、法律法规、真实流程、合规常识 |
| **时效性** | 定期更新，同步当地政策、规则变动 |
| **实用性** | 聚焦用户实操问题，杜绝冗余理论内容 |
| **本地化** | 完全贴合目标国家本地规则，无通用泛化内容 |

### 6.3 技术核心要求

| ID | 要求 | 指标 |
|----|------|------|
| TF-01 | 多库路由分发 | 用户点击顾问卡片 → 自动匹配国家+角色专属知识库 |
| TF-02 | 检索精准度 | 答案优先来源于本地专属知识库，准确率 ≥ 95% |
| TF-03 | 响应速度 | 单机部署下，问答响应时间 ≤ 3 秒 |
| TF-04 | 迭代便捷性 | 新增国家/语种/顾问 → 仅需新增知识库，无需改动系统底层 |

## 七、真人顾问对接模块需求

### 7.1 顾问入驻要求

- 具备目标国家本地专业资质（留学顾问、移民持牌、律师、职场导师等）
- 熟悉当地跨境相关政策与实操流程
- 提交真实资质证明，通过平台审核

### 7.2 业务流程

```
用户 AI 咨询 → AI 无法解答 → 系统推送当地真人顾问
    → 用户预约/付费 → 线上一对一咨询 → 订单完成/评价
```

### 7.3 服务模式

| 模式 | 说明 |
|------|------|
| 免费 AI 咨询 | 基础自助咨询，无次数限制（或高频限流） |
| 付费真人咨询 | 单次咨询 / 套餐咨询 / 定制规划服务 |
| 咨询形式 | 线上文字 / 语音 / 视频 多种形式 |

## 九、产品核心壁垒

1. **通用化架构** — 全球唯一一套架构适配全国家、全语种，复制成本极低
2. **全场景覆盖** — 唯一覆盖跨境全流程咨询的 AI 平台，无服务盲区
3. **技术 + 实操结合** — 懂技术可自主搭建 RAG 系统，懂跨境实操可打造精准知识库
4. **去中心化服务** — 摆脱传统人工中介依赖，实现标准化普惠服务
5. **无限迭代能力** — 知识持续沉淀，平台越用越专业，竞争壁垒持续加高

## 十、风险与规避

| 风险 | 概率 | 缓解措施 |
|------|------|---------|
| 政策合规风险 | 中 | 知识库仅输出官方公开信息，不提供法律/移民代办服务，真人顾问持证上岗 |

| 多语种质量风险 | 中 | 使用 multilingual embedding + 母语审校 |
| 真人顾问供给风险 | 中 | 阶段二才启动，先用 AI 验证需求，再引入真人 |
